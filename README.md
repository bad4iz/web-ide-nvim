# Web ide на nvim
## Сборка nvim для Web разработки.


## Installing Windows
```bash
git clone https://github.com/bad4iz/web-ide-nvim $env:LOCALAPPDATA\nvim
nvim
```

### Переключение языка `Ctrl` + `6`
